8.6 SML#のランク1多相性

問 8.10

本節の冒頭で与えた fとgの （ランク1多相性を持つ型システムの下での） 型を予想し， SML#で確認せよ．

解答例  以下は、SML#での評価結果である。

   # val f = ((fn x => x) 1, fn x => fn y => (x + 1,ref y));
   val f = (1, fn) : int * (int -> [’a. ’a -> int * ’a ref])
   # val g = (#2 f) 1;
   val g = fn : [’a. ’a -> int * ’a ref]

問 1.6

第1.2節で作成した3文字を比較する プログラムをファイルtry.smlに作成し，実行せ よ．

解答例  
ファイルtry.sml：

   if #"?" < #"$" then
      if #"$" < #"*" then
        (str #"?") ^ (str #"$") ^ (str #"*")
      else if #"?" < #"*" then
        (str #"?") ^ (str #"*") ^ (str #"$")
      else (str #"*") ^ (str #"?") ^ (str #"$")
   else if #"?" < #"*" then
     (str #"$") ^ (str #"?") ^ (str #"*")
   else if #"$" < #"*" then
     (str #"$") ^ (str #"*") ^ (str #"?")
   else (str #"*") ^ (str #"$") ^ (str #"?") ;

実行結果

$ smlsharp
# use "try.sml";
val it = "$*?" : string

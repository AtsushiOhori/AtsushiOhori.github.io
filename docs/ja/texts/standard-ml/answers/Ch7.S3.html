7.3 パターンマッチングの拡張機能

問 7.8

上記 f、つまり、

   fun f Empty = true
     | f (Node(_,Empty,Empty)) = true
     | f (Node(_,x as Node _, y as Node _)) = f x andalso f y
     | f _ = false

が true を返す2分木はどのような木か？

解答例  Node(_, Empty, Node _)の形のノードを含まない木、つまり、完全２分木である。

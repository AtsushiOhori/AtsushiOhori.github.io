ディジタルコンピュータの構造と動作原理、コンピュータ上の最初の大規模なソフトウェアであるＯＳを構成する基本概念と機能を解説します。大学の情報科学関連の学部レベルの講義８回分に相当する内容ですが、コンピュータに興味を持つ方ならどなたでも、予備知識なしに、コンピュータでの問題解決の本質，さらに，ハードウエアと協調して動くOSの構造と機能が理解できる内容です．
